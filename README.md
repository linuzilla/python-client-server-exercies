# python-client-server-exercies
Just for fun!

## Get Started

```sh
$ virtualenv -p python3 venv
$ source venv/bin/activate
(venv) $  pip install --upgrade pip
(venv) $  pip install -r requirements.txt
```

## To run the server
```sh
(venv) $  python sample_server.py
```

## To run the client
